        <div class="subject_summary" id="subject_summary" property="v:summary">
         铠V，高中3年级，双簧管演奏者。
         <br/>
         伞木希美，高中3年级，长笛演奏者。
         <br/>
         <br/>
         初中时，是希美牵起了的手，带领她走出了孤独。
         <br/>
         自那以后，希美就占据了的整个世界。
         <br/>
         只有与希美在一起的生活，才令感到幸福。
         <br/>
         可在高中一年级，希美一度退出了吹奏部。而她的归来，也始终未能抹去心中的不安。
         <br/>
         ――是否有一天，希美会再次消失在自己面前呢？
         <br/>
         <br/>
         就这样，二人迎来了高中最后的竞演会。
         <br/>
         参赛的自由曲是「莉兹与青鸟」。
         <br/>
         在这首以童话为蓝本创作的乐曲当中，有一段双簧管与长笛交相呼应的章节。
         <br/>
         <br/>
         「故事的结局，还是越美满越好啦。」
         <br/>
         虽然希美如此认为，却依然对终将到来的离别心怀恐惧。
         <br/>
         <br/>
         ――请永远，永远，留在我身边吧――
         <br/>
         <br/>
         在继续着日常生活的同时，两人总是情不自禁地将自己投影到童话当中。
         <br/>
         是莉兹，希美是青鸟。
         <br/>
         但是……
         <br/>
         两枚略有嫌隙的齿轮，仍不停地转动，寻找着与彼此相契合的那个瞬间。
